Scientific notation calculator converts numbers to scientific notation, e-notation, engineering notation, and adds, subtracts, multiplies or divides them.

[https://www.calculator.io/scientific-notation-calculator/](https://www.calculator.io/scientific-notation-calculator/)

== Usage ==

Add the Scientific Notation Calculator shortcode to your page, post or sidebar:

`[ci_scientific_notation_calculator]`

Add the following code to your template where you would like the Scientific Notation Calculator to appear:

`<?php display_ci_scientific_notation_calculator(); ?>`

== Installation ==

1. Upload the Scientific Notation Calculator /ci_scientific_notation_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Scientific Notation Calculator plugin through the "Plugins" menu in WordPress.
